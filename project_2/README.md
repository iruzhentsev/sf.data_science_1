# Проект 2: Анализ вакансий HeadHunter

## 📌 Описание
Данный проект представляет собой комплексный анализ базы вакансий с сайта HeadHunter с использованием SQL (через Python) и библиотек для обработки данных. Основная цель — исследовать рынок IT-вакансий, выявить ключевых работодателей, востребованные навыки и зарплатные тенденции, с фокусом на направление Data Science. Результаты могут быть полезны как соискателям для планирования карьеры, так и HR-специалистам для понимания рыночной ситуации.
Проект выполнен в рамках курса Data Science от Skillfactory.

## 📁 Содержание
- `Skillfactory_Project_2_Headhunter_Руженцев.ipynb` — Jupyter Notebook с полным анализом (SQL-запросы, результаты, выводы, графики).
- `README.md` — описание проекта и инструкция (текущий файл).

## 🛠 Используемые технологии
- **Python** 3.10+
- **Pandas** — обработка и анализ данных
- **Psycopg2** — подключение к PostgreSQL и выполнение SQL-запросов
- **Matplotlib** и **Seaborn** — визуализация результатов
- **Requests** и **lxml** — парсинг данных из Википедии (список городов-миллионников)
- **PostgreSQL** — база данных с вакансиями (учебная)

## 🚀 Запуск проекта
### Локальный запуск
1. Клонируйте репозиторий (если ещё не сделали):
   ```bash
   git clone https://github.com/iruzhentsev/sf.data_science_1.git
   cd sf.data_science_1
2. Установите необходимые библиотеки из корневого requirements.txt:
   bash
   pip install -r requirements.txt
3. Перейдите в папку проекта и откройте ноутбук:
   bash
   cd project_2
   jupyter notebook Skillfactory_Project_2_Headhunter_Руженцев.ipynb

**Важно:** для выполнения SQL-запросов необходимы параметры подключения к учебной базе данных Skillfactory. Они **не включены** в публичную версию ноутбука. Перед запуском вставьте свои данные в ячейку с параметрами (логин, пароль, хост, порт, название базы). Эти данные можно получить на платформе курса.

## 📊 Основные результаты и выводы
•	**Общая характеристика:** база содержит 49 197 вакансий, 23 501 работодателя, 1 362 региона. Большинство вакансий сосредоточено в Москве, Санкт-Петербурге и других крупных городах.
•	**Работодатели:** лидер по числу вакансий — Яндекс (1933). В сегменте Data Science наибольшее число вакансий у СБЕРа (39).
•	**Зарплаты:** средняя зарплата DS с опытом 3–6 лет — 256 454 руб., для начинающих — 74 643 руб.
•	**Ключевые навыки:** Python и SQL — самые востребованные. В среднем в DS-вакансиях указывается 6,55 навыков.
•	**Рынок для начинающих:** доступно всего 51 вакансия для Junior DS, что свидетельствует о высоком пороге входа.
Подробные цифры, графики и поэтапные выводы представлены в ноутбуке.
