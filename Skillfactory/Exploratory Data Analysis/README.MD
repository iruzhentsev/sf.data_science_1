# Исследование данных о зарплатах в Data Science

## Описание проекта
HR-агентство изучает тренды на рынке труда в IT. В рамках исследования проведён анализ данных о зарплатах специалистов в сфере Data Science за 2020–2022 годы (датасет [Data Science Job Salaries](https://www.kaggle.com/datasets/ruchi798/data-science-job-salaries)).

## Цели исследования
- Выяснить, какие факторы влияют на зарплату специалиста Data Scientist.
- Ответить на ключевые вопросы HR-агентства:
  - Наблюдается ли ежегодный рост зарплат у специалистов Data Scientist?
  - Как соотносятся зарплаты Data Scientist и Data Engineer в 2022 году?
  - Как соотносятся зарплаты специалистов Data Scientist в компаниях различных размеров?
  - Есть ли связь между наличием должностей Data Scientist и Data Engineer и размером компании?
- Провести статистическое тестирование для подтверждения выводов.

## Структура ноутбука
1. Загрузка и обработка данных
2. Разведывательный анализ данных
3. Статистический анализ (тесты для одной, двух и нескольких выборок, тест для категориальных признаков)
4. Дополнительные гипотезы
5. Итоговые выводы

## Используемые библиотеки
Все библиотеки перечислены в файле `requirements.txt`.

## Результаты
- Подтверждён ежегодный рост зарплат Data Scientist.
- Не обнаружено значимых различий между зарплатами Data Scientist и Data Engineer в 2022 году.
- Размер компании и уровень опыта значимо влияют на зарплату.
- Связи между типом должности и размером компании не выявлено.
